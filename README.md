# Running this tool from your computer..
### Phase1 - Cloning the Repo
git clone https://github.com/andrefernandes86/demo-txone-tcpreplay.git

### Phase2 - Installing the Application
./lx_install.sh

### Phase3 - Adjusting the parameters
Open the file "pcaps.sh" and map to the directory where you downloaded the "demo-txone" repo

### Phase4 - Sending the PCAPs over the network
Execute the script: "./pcaps.sh"
